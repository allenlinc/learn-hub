分数、小数、百分数互化

班级投票：喜欢公园的占 3/4，喜欢博物馆的占 0.7，哪个更多？今天学会把三种"语言"变统一。

百分数，表示一个数是另一个数的百分之几的数，叫做百分数；分数、小数与百分数，只是同一个数的不同写法。

Concept 1

分数 ⇄ 小数

分数和小数说的是同一类"部分与整体"，差别只在写法。

分数 → 小数
  • 分母是 10、100、1000：直接写，看分母几个 0。如 7/10 = 0.7，13/100 = 0.13。
  • 一般分数：分子 ÷ 分母。1/4 = 1÷4 = 0.25；3/8 = 0.375。
小数 → 分数
  • 看有几位小数，1 后写几个 0 作分母，去小数点作分子，再约分。
  • 0.6 = 6/10 = 3/5；0.125 = 125/1000 = 1/8。
⚠️ 易错 err-2：分数化小数别搞反——是分子÷分母（1÷4），不是 4÷1，也不是 0.14。
Concept 2

小数 ⇄ 百分数

小数 → 百分数

小数点向右移两位，加 %。

0.25 → 25%　|　1.2 → 120%

百分数 → 小数

去 %，小数点向左移两位。

75% → 0.75　|　120% → 1.20

⚠️ 易错 err-1：小数化百分数要右移两位（0.25→25%），不是左移成 2.5%。
⚠️ 易错 err-3：120% 去 % 左移两位是 1.20，不是 0.012，也不是 120。
Concept 3

分数 ⇄ 百分数

最稳的办法：先化小数，再化百分数。

1
分数 → 小数：分子÷分母。如 1/4 = 0.25。
2
小数 → 百分数：右移两位加 %。0.25 → 25%。所以 1/4 = 25%。
3
反过来：百分数 → 分数，先化小数再化分数并约分。75% = 0.75 = 75/100 = 3/4。
💡 约分不成整数的百分数（如 1/3 ≈ 33.3%）用"约等于"，课标只要求会化、不必把循环小数化成分数。
Worked Example

例题：比较并排序

把 3/4、0.72、75% 化成同一种形式，按从大到小排列。ex-1

1
统一化成小数最方便：3/4 = 3÷4 = 0.75；75% = 75÷100 = 0.75；0.72 不变。
2
比较：0.75 = 0.75 > 0.72。
3
结论：3/4 = 75% > 0.72。两种写法其实相等！
🎯 关键：比较前先统一计数单位（都化小数）。这正是整数、分数、小数加减运算都要"相同计数单位"的一致思想。

Summary

一页记住

①
分数→小数：分子÷分母（1/4=0.25）。
②
小数→百分数：右移两位加 %（0.25→25%）；反过来去 % 左移两位。
③
分数→百分数：先化小数再化百分数，或化成分母 100 的分数。
④
比较大小：先统一化成小数（统一计数单位）再比，最稳。
🌱 分数、小数、百分数只是同一个数的三种写法。互化的本质是统一计数单位——这正是整数、分数、小数运算一致性的起点。
记忆锚点

🧠 口诀卡

分变小数，分子÷分母；小变百分数，右移两位加 %；百分数变小，去号左移两位准。

深层理解

🔍 五镜头看本质

  • 看见它：分数、小数、百分数只是同一个数的三种"外衣"。
  • 拆开它：分数=分子÷分母；小数=十进制位值；百分数=分母 100。
  • 解释它：互化的本质是统一计数单位。
  • 比较它：都化成小数再比，最稳。
  • 迁移它：折扣、命中率、出勤率都用百分数表达。
