# 学科学习网站 (Study Website)

一个面向多科目的学习资源站点，按学科组织内容，方便逐步填充资料、练习与笔记。

## 学科导航

| 学科 | 文件夹 | 说明 |
|------|--------|------|
| 语文 Chinese | `chi/` | 阅读理解、写作、古诗文 |
| 英语 English | `eng/` | 词汇、语法、阅读 |
| 数学 Math | `math/` | 算术、代数、几何 |
| 物理 Physics | `phy/` | 力学、电学、光学 |
| 化学 Chemistry | `chem/` | 物质、反应、实验 |
| 生物 Biology | `bio/` | 细胞、遗传、生态 |
| 历史 History | `hist/` | 中国史、世界史 |
| 地理 Geography | `geo/` | 自然地理、人文地理 |
| 政治 Politics | `pol/` | 道德与法治、国情 |

## 本地预览

直接用浏览器打开 `index.html` 即可看到学科导航页。

## 目录约定

- 每个学科一个文件夹，使用英文缩写命名（避免 URL 编码问题）。
- 各文件夹内放置该学科的 `README.md` 作为入口说明。
- 后续可按知识点再分子目录，例如 `math/xiaoshu/`（小数）。

## 贡献

欢迎补充各学科资料，提交 PR 或在本地直接编辑后推送。
